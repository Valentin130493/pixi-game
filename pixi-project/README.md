# Project

## Description
PIXI game

## Installation
1. Clone the repository
2. Install the dependencies by running `npm install`

## Usage
1. Start the project in development mode by running `npm run dev`
2. Open your browser and navigate to `http://localhost:8080` (or the specified URL)
3. Make changes to your code and see them live in the browser

## Building
1. To build the project for production, run `npm run build`
2. The optimized files will be generated in the `dist` directory

## Linting
1. To run the linter and check your code against the specified rules, run `npm run lint`

## Dependencies
- `pixi.js`: The main package for working with graphics
- `eslint`: A code linter for checking code quality and enforcing coding standards
- `prettier`: A code formatter for maintaining consistent code style
- `webpack`: A module bundler for managing project dependencies and assets
- `typescript`: A programming language that adds static typing to JavaScript
- `typescript-eslint`: An ESLint plugin for TypeScript
- `webpack-cli`: The command-line interface for webpack
- `webpack-dev-server`: The development server for webpack
- `copy-webpack-plugin`: A webpack plugin for copying files to the output directory
- `html-webpack-plugin`: A webpack plugin for generating HTML files
- `terser-webpack-plugin`: A webpack plugin for minifying JavaScript code
- `ts-loader`: A webpack loader for TypeScript files
- `ajv`: A JSON schema validator
- `ajv-keywords`: A package for adding custom keywords to JSON schemas
